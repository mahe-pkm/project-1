# Generic Website QA Checklist

## Visual

```txt
[ ] Layout matches requirement/reference
[ ] Typography is readable
[ ] Spacing is consistent
[ ] Buttons look clickable
[ ] Sections are visually clear
[ ] Footer is complete
```

## Responsive

```txt
[ ] 1440px desktop
[ ] 1280px laptop
[ ] 1024px tablet landscape
[ ] 768px tablet
[ ] 430px mobile
[ ] 390px mobile
```

## Functionality

```txt
[ ] Navigation works
[ ] Links work
[ ] Buttons work
[ ] Forms have labels
[ ] Form validation is clear
[ ] Modals/menus close properly
[ ] No broken images
```

## Accessibility

```txt
[ ] One logical H1
[ ] Images have alt text
[ ] Buttons have accessible names
[ ] Links have meaningful text
[ ] Keyboard focus is visible
[ ] Color contrast is readable
```

## SEO Basics

```txt
[ ] Title tag
[ ] Meta description
[ ] Logical heading structure
[ ] Descriptive image alt text
[ ] Clean URLs/file names
```

## Git Readiness

```txt
[ ] No secrets
[ ] No backup folders
[ ] No temporary files
[ ] Asset map updated
[ ] Docs updated where needed
```
